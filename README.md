Hej og velkommen til LONERS!

For at kunne eksekvere webapplikationen skal nedenstående først køres i terminalen:
- npm install
    - Evt også hent sqlite, hvis den driller
    - npm install sqlite3
- Projektet kan hernæst køres med: node / nodemon index.js
